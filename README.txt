[바로 실행용 최종본]
관리자 로그인 포함 버전입니다.
관리자 비밀번호는 muan0346 입니다.

실행 방법
1. 압축 풀기
2. 해당 폴더에서 cmd 열기
3. 아래 순서대로 입력

npm install
npm start

접속 주소
- 메인: http://localhost:3000
- 관리자 로그인: http://localhost:3000/admin/login

카테고리
- 절도예방수칙
- 사기예방수칙
- 외국인 범죄예방수칙
- 무안경찰알림

참고
- 이 압축파일에는 .env 파일이 이미 들어있습니다.
- Render 배포 후 관리자 로그인 주소는 /admin/login 입니다.
